About this task

Estimated time 2 hrs

Goal

The aim of this task is to evaluate a number of techniques frequently used in Angular 2

These include data manipulation, service calls and use of components.

The goal is not to see how much you know about angular but how you solve the problem, research and refer to other examples and explain your decision making in code comments where possible.

The tasks tab will contain your solution.

The solution should include

  • A button to call data from the rest service.
  • A list of the data including the name and social media platform, sorted into size order.
  • The data does not need to sortable beyond this.
  • A user pressing the button to call data should get new data and display it.
  • BONUS - A user should be able to filter the list of data by name.

Hints and Tips

  • Don't change any code in the rest service, assume no errors are thrown from here either.
  • Don't worry about using typescript types or applying css styles
  • All components and services required should be present within the task folder.
  • Investigate using promises, observables and component pipes.
  • Consider explanations for your choices thinking about performance, stability
  • The components set up should already include the required imports.
  • Feel free to use online resources. This is a useful starting point.
